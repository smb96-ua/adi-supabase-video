DESCRIPCIÓN
-----------
App móvil híbrida con Ionic 8 + Vue 3 para gestionar películas y series.
CRUD completo sobre ambos recursos con interfaz móvil nativa.

TECNOLOGÍAS: Ionic 8, Vue 3, Capacitor 7, PocketBase, TypeScript

ESTRUCTURA DEL PROYECTO
-----------------------
/
├── backend/                       # PocketBase (ejecutable + datos)
├── src/                           # Código fuente Vue/Ionic
│   ├── backend/                   # Configuración y repositorios
│   └── views/                     # Páginas (Login, Movies, Series, Tabs)
├── public/                        # Assets PWA (manifest, service-worker)
└── tests/                         # Tests E2E y unitarios

================================================================================
                          INSTRUCCIONES DE EJECUCIÓN
================================================================================

1. Iniciar PocketBase:    cd backend && ./pocketbase serve
2. Instalar dependencias: npm install
3. Ejecutar app:          npm run dev
4. Acceder:               http://localhost:5173
